# 색상 추가요금(priceDelta) 백필 정리

작성일: 2026-06-09

## 1. 배경 — 무엇이 누락되어 있었나

carpan2 원본 데이터(`raw_data_ko_car.json`, `raw_data_imported_car.json`)에는 색상 추가요금이
**두 위치**에 존재한다.

- **모델 단위**: `model.colorExt` / `model.colorInt` (TSV `색상id\t가격\t플래그`)
- **트림 단위**: `trim.colorExt` / `trim.colorInt`

기존 `scripts/import-vehicles.ts`는 **모델 단위 가격만** `VehicleColor.priceDelta`로 저장하고,
트림 단위 가격은 `detailedSpecs`에만 보존한 채 무시했다. 그 결과 **트림 단위에만 가격이 있는 차량**은
DB에 0원으로 들어가, 고객 견적 화면에서 전부 "추가요금 없음"으로 표시됐다.

- 보정 전: 추가요금 있는 차량 **27대**
- 원본 기준 실제로 추가요금이 있는 차량: 국산·수입 합쳐 약 90대

## 2. 적용 완료 내용 (국산)

국산 데이터(`raw_data_ko_car.json`)는 트림 단위 가격이 깨끗하고 균일(8~50만원, 실제 도색 추가요금과 일치)하여
백필했다.

| 항목 | 값 |
| --- | --- |
| 갱신된 색상 | **70개** |
| 대상 차량 | **34대** (싼타페, 투싼, 팰리세이드, 더 뉴 그랜저, K5, 카니발, 셀토스, EV4, 아이오닉 9, 캐스퍼 일렉트릭 등) |
| 금액 범위 | 8~50만원 |
| 결과 | 추가요금 차량 27 → **59대**, 추가요금 색상 99 → **168개** |

### 처리 방식

- **병합 규칙**: 색상 externalId 기준으로 모델 단위 + 모든 트림 단위 가격 중 **최댓값**.
  - 트림마다 가격이 다른 소수 케이스(예: 르노 그랑 콜레오스, 현대 팰리세이드)는 `VehicleColor`가
    차량당 단일 값(per-vehicle)이라 트림별 차등을 담을 수 없어 **최댓값으로 통일**(과소청구 방지).
- **기존값 보호**: 계산값이 기존 DB 값보다 **클 때만** 갱신. 어드민에서 수동 입력한 값
  (예: 테슬라 New Model Y Pearl White 100만원)은 보존된다.
- **멱등성**: 재실행 시 추가 갱신 0건.

### 코드 변경

1. **`scripts/import-vehicles.ts`** — 앞으로 재임포트 시 국산차 트림 단위 색상가를 모델 단위로 자동 병합
   (`mergeTrimColors` 플래그, 국산 파일만 `true`).
2. **`scripts/backfill-color-prices.ts`** (신규) — 현재 DB 보정용. 기본 dry-run, `--apply`로 반영.
   ```
   pnpm tsx scripts/backfill-color-prices.ts            # 미리보기
   pnpm tsx scripts/backfill-color-prices.ts --apply    # DB 반영
   ```
3. **`src/components/admin/vehicles/edit/ColorManager.tsx`** — 색상 추가요금 입력 단위를 **원 → 만원**으로 변경
   (예: `100` 입력 = 100만원). 저장 시 `× 10000`, 표시 시 `÷ 10000`. API·DB는 원 단위 유지.

## 3. 보류 — 수입차 색상 추가요금 (29대)

수입차(`raw_data_imported_car.json`)는 **트림마다 색상가가 제각각이고 극단적**이라 자동 백필에서 제외했다.

대표 사례:

| 브랜드 | 모델 | 트림별 색상가 범위 |
| --- | --- | --- |
| 벤츠 | Maybach EQS SUV | 247만 ~ **3,580만** |
| 벤츠 | Maybach S-Class | 247만 ~ 3,171만 |
| 마세라티 | Grecale | 170만 ~ 2,580만 |
| 랜드로버 | The New Range Rover | 165만 ~ 1,815만 |
| 벤츠 | The New E-Class | 42만 ~ 860만 |
| 테슬라 | New Model Y | 128만 ~ 275만 (Launch 트림은 0원, RWD 트림은 유료) |

### 문제점

1. **트림 종속성**: 테슬라처럼 같은 색이 트림에 따라 무료/유료가 갈린다. per-vehicle 단일 값으로는
   최댓값을 잡으면 무료 트림 고객에게 과청구된다.
2. **극단값**: 마세라티 비스포크(Fuoriserie), 벤츠 마이바흐 Manufaktur 도색은 실제로 수백~수천만 원이라
   파싱 오류가 아닌 원본 실제값이지만, 대중 견적 흐름에 그대로 노출하기엔 부적절하다.

### 권장 처리 방안

- **단기(권장)**: 현 상태 유지(수입 럭셔리는 색상 추가요금 0원 표기). 해당 차종은 어차피 **개별 상담** 견적이
  일반적이므로, 필요한 모델만 어드민에서 **대표값을 수동 입력**한다.
- **중기**: 자주 견적되는 수입 모델(예: 테슬라, BMW)만 선별해, 기본/최저가 트림 기준 대표 색상가를
  어드민에 수기 등록.
- **장기(필요 시)**: 색상 추가요금을 트림 단위로 모델링(스키마 변경 + 어드민 재설계)해야 정확하지만,
  적용 대상이 극소수라 비용 대비 효율이 낮다. 현재는 비권장.
