### 1. 장기렌트 / 리스 상품 유형별 회수율 분리

`CapitalRateSheet`에 `productType String @default("장기렌트")` 추가. 기존 unique constraint를 `(financeCompanyId, trimId, weekOf, productType)` 으로 확장해 동일 트림·주차에 장기렌트/리스 데이터를 별도 관리.

- 회수율 입력 화면 상단에 **장기렌트 / 리스** 토글 추가
- 견적 시뮬레이터에 상품 유형 토글 추가 — 선택에 따라 해당 회수율 데이터로 계산
- API GET에 `?productType=` 쿼리 파라미터 필터 추가

### 2. 관리자 권한 3단계 재정의

| 역할 | 권한 |
|------|------|
| 최종관리자 (`superadmin`) | 모든 권한 — 권한 부여/회수 + 콘솔 접근 + 데이터 수정 |
| 관리자 (`admin`) | 권한 부여/회수 불가, 콘솔 접근 가능 |
| 딜러 (`dealer`) | 콘솔 접근 불가 (레이아웃에서 `/` 리다이렉트) |

- 사용자 상세 패널의 권한 버튼을 최종관리자에게만 렌더링
- 관리자는 역할을 읽기 전용으로만 확인 가능
- 사이드바 하단에 현재 로그인 계정의 역할 배지 표시

### 3. 할인가 견적 계산 반영

`discountPrice`가 설정된 트림은 모든 견적 계산 경로에서 `discountPrice`를 차량가로 사용. 고객 화면에는 원가 취소선 + 할인 금액 배지로 표시.

### 4. 회수율 외삽(Extrapolation) 적용

기존 min/max 범위를 벗어난 차량가에 대해 클램프(고정) 대신 선형 연장선상에서 회수율 계산. 옵션 추가 등으로 범위가 초과될 때 왜곡 없이 처리.

### 5. 보증금/선납금 입력 단순화

보증금 10%당 가산율, 선납금 10%당 가산율은 계약 조건(개월·거리)에 무관하게 동일. 9개 조합을 모두 입력하는 대신 **36개월·1만km 단일 셀**만 입력하도록 제한, 나머지는 음영 처리.

### 6. 트림 검색 키워드 방식 개선

기존 단순 `includes` → 공백 기준 토큰 분해 + 한글↔숫자 경계 분해. 입력 순서·공백 여부에 무관하게 키워드가 모두 포함되면 매칭.
